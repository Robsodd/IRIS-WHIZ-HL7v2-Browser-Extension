Instances & Namespaces

Use the table below to add your instances and namespaces to the Iris Whiz extension.

Instances:

Instances are required for most features to work.

The Name field will be how your instance appears in the Tab groups.

The URL field should be your DNS, with no leading or trailing characters.

E.g. my.dns.example and NOT https://my.dns.example/


Click the namespaces section to add namespaces.
The Colour field is how you choose your header and tab group colour. If using a custom colour the tab group will default to grey.

Namespaces:

The Namespaces are required for some features to work.


Custom Colours

Use the table below to add your custom colours to the Iris Whiz extension.


Adding a custom colour with the same name as a default colour will overwrite that default colour.

Tab group colours are limited to the Browser's default list and are selected by selecting one of the default colours.

Selecting a custom colour will result in a grey tab group.

There is a backup background colour in case you want to use a gradrient background colour and share your Iris Whiz setup across browsers. Set the primary backup to the gradient you want and the fallback to the block colour.
